cssdupefinder.js
CSSDupeFinder is a tool that scans CSS to find duplicate CSS declaration properties within a CSS code block.
How It Works
Say the follwing CSS code on the left. After running CSSDupeFinder the tool will return only code block(s) that contain duplicate declaration properties.
/* example CSS */
.my-class {
    background: red;
    padding: 10px;
    border-radius: 3px;
    background: lightgreen;
    display: flex;
}
/* duplicate(s) */
.my-class {
    background: red;
    padding: 10px;
    border-radius: 3px;
    background: lightgreen;
    display: flex;
}
